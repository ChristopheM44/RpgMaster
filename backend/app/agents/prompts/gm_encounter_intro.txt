Tu dois produire l'ouverture cinematographique de la rencontre, juste avant le premier tour d'initiative.

Objectif :
- Decrire la scene de combat en 3 a 4 phrases immersives : lumiere, reliefs tactiques, objets saillants, lignes de fuite.
- Donner une silhouette reconnaissable aux ennemis, et identifier un meneur si cela semble naturel.
- Inclure une posture tactique claire : arme levee, embuscade, repli, sommation, panique, prise d'otage, terrain favorable, etc.
- Ajouter au moins une replique courte d'ennemi entre guillemets francais, sauf si les ennemis sont incapables de parler.
- Si la rencontre se deroule dans un nouveau lieu structure ou si game_state.current_scene est absent, ajoute UNE action scene_layout pour que cette scene reste visible apres le combat.
- Dans scene_layout, ajoute description/action_hint courts aux points d'interet tactiques, dangers, couverts ou passages, afin que la carte explique ce que le joueur selectionne.
- Categories visuelles obligatoires dans scene_layout : ennemi visible = kind "enemy", icon "c-enemy"; PNJ non hostile = kind "npc", icon "npc"; couvert = kind "cover", icon "c-half-cover"; danger/piege = kind "hazard", icon "trap-danger"; butin/objet = kind "loot", icon "chest"; indice = kind "clue", icon "clue".
- Une porte, issue ou sortie navigable doit etre dans exits uniquement. Ne la duplique pas dans pois avec le meme id ou la meme position.
- Pour un PNJ ou objet important visible sur la carte, tu peux ajouter interactions (label, intent: approach|talk|examine|listen|search|use|custom, prompt optionnel).
- Ne demande pas de jet, ne modifie aucune statistique, ne relance pas encounter_setup, et ne change pas la phase.
- Ajoute "start_mode": "pause" si les ennemis somment, menacent, parlementent, hesitent ou laissent une chance de reagir.
- Ajoute "start_mode": "combat" si les ennemis attaquent immediatement, chargent, tirent ou si l'embuscade est deja declenchee.

Ennemis generalement muets ou non parlants :
- skeleton, zombie, wolf, giant_spider
- squelettes, zombies, loups, araignees geantes, betes sauvages, morts-vivants sans intelligence

Si les ennemis sont muets, remplace la replique par un signal narratif equivalent : claquement d'os, grondement, sifflement, hurlement, raclement d'armes.

Format attendu :
- Reponds avec UN SEUL objet JSON conforme au system prompt.
- "actions" doit etre [] sauf pour une unique action scene_layout.
- "mood" doit refleter la tension de la scene.
- "start_mode" doit etre "pause" ou "combat".
- Aucun texte hors JSON.
